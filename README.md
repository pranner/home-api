## Description

Home Api test code Sample. Here you will see a few snippets of code that can be used to create an API to connect homebuyers to homes.

## Requirements

[Node.js version 12.14.X (includes npm 6.13.X)](https://nodejs.org/en/download/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
